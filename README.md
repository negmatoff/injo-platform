# 🔗 injo.me/injo.pro

Link shortener and bio pages platform for Central Asia

## 🚀 Features

- Short links with analytics
- QR code generation
- Link-in-bio pages
- P2P payments integration

## 🛠️ Tech Stack

- Next.js 14 + TypeScript
- Tailwind CSS + Headless UI
- Prisma + PostgreSQL (Supabase)
- NextAuth.js v5
